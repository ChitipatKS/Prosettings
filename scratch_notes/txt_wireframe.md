# Txt Wireframe

#### 🏆 โครงสร้างหน้าแรก (Home Page) พร้อมระบบ Ranking
**1. Hero Section & Smart Search (ส่วนบนสุด - ดึงดูดสายตา)**
  - เปิดเว็บมาปุ๊บ สิ่งแรกที่เจอคือช่องค้นหาขนาดใหญ่ (Smart Search Bar) ที่พิมพ์ข้อความอิสระได้ ไม่ว่าจะพิมพ์ชื่อโปรเพลเยอร์, สังกัดทีม, หรือแม้กระทั่งชื่อรุ่นอุปกรณ์
  - ด้านหลังช่องค้นหา อาจจะเป็นภาพพื้นหลังเท่ๆ ของโปรเพลเยอร์ที่กำลังเป็นอันดับ 1 ในสัปดาห์นั้น
**2. The "Meta" Leaderboard (ส่วนจัดอันดับ Pro Player)**
  - นี่คือส่วนที่คุณเสนอมาครับ เราสามารถทำเป็นตาราง Leaderboard โชว์ว่า Setting ของใครฮิตสุด โดยระบบจะนับคะแนน (Usage Count) จาก:
    - จำนวนครั้งที่มีคนกดปุ่ม "Copy to Clipboard" สำหรับโค้ด Crosshair, โค้ด Viewmodel หรือตาราง Config (CFG)
    - จำนวนครั้งที่ผู้ใช้ทั่วไปดึง Setting ของโปรคนนั้นไปเป็นฐานในการสร้าง Community Configs ของตัวเอง
  - เราอาsดฮิตจาก Community" ตีคู่ไปกับพื้นที่ของ Pro Player เลยครับ
  - เปิดโอกาสให้ผู้ใช้ทั่วไปสามารถโหวตให้คะแนน Settings ของคนอื่นได้ เพื่อสร้างระบบจัดอันดับ (Leaderboard) ของคนธรรมดาให้โดดเด่นขึ้นมาครับ
**4. Trending Gear (ส่วนทำเงิน)**
  - โชว์อุปกรณ์ Gaming Gear ที่กำลังมาแรงที่สุดในสัปดาห์นั้น (อิงจากสถิติที่คนค้นหาและกดดูบ่อย) พร้อมปุ่มกดซื้อเพื่อรับค่าคอมมิชชัน
การจัดวางแบบนี้จะทำให้หน้าแรกของเว็บดูมีชีวิตชีวา (Dynamic) และอัปเดตข้อมูลแบบ Real-time ตามพฤติกรรมของคนใช้งานจริงๆ ครับ

#### 🔍 โครงร่างหน้าค้นหา (Search Section Text Wireframe)
**[ Smart Search Bar ]**
  - ช่องค้นหาขนาดใหญ่ พิมพ์ข้อความอิสระได้ ไม่ว่าจะพิมพ์ชื่อโปรเพลเยอร์, สังกัดทีม, หรือแม้กระทั่งชื่อรุ่นอุปกรณ์  (เช่น พิมพ์ "TenZ" หรือ "Superlight")
**[ แผงตัวกรองด่วน (Quick Filters) ]**
เราจะจัดวางปุ่มตัวกรองเป็นหมวดหมู่ใต้ช่องค้นหา เพื่อให้กดใช้งานได้ทันที:
  - **หมวดเกม (Games):**
    - [ Valorant ] | [ CS2 ] | [ Apex Legends ] | [ Overwatch ]
  - **หมวดตำแหน่ง (Roles - *****จะเปลี่ยนไปตามเกมที่เลือกด้านบน*****):**
    - *ถ้าเลือก Valorant:* [ Duelist ] | [ Initiator ] | [ Controller ] | [ Sentinel ]
    - *ถ้าเลือก CS2:* [ AWPer ] | [ Rifler ] | [ IGL ] | [ Entry Fragger ] ตัวอย่างเช่น กรองตามบทบาทในทีม (เช่น Duelist ใน Valorant หรือ AWPer ใน CS)
  - **หมวดอุปกรณ์ (Gaming Gears):**
    - [ Mouse ] | [ Keyboard ] | [ Mousepad ] | [ Headset ] โดยเราจะเน้นเฉพาะอุปกรณ์เหล่านี้ และจะไม่ดึงส่วนของราคาประเมินของ PC Spec มาเป็นตัวกรองครับ

#### 💡 การแสดงผลลัพธ์ (Search Results)
เมื่อผู้ใช้พิมพ์ค้นหา หรือกดเลือกตัวกรอง (เช่น กดเลือก *Valorant* + *Duelist* + *Mouse*) ด้านล่างของจอจะแสดงผลลัพธ์เป็น **"การ์ดโปรไฟล์"** ขนาดย่อเรียงกันครับ โดยในการ์ดย่อนี้ควรมี:
  - รูปโปรไฟล์ และชื่อผู้เล่น
  - สังกัดทีม และโลโก้เกม (เช่น โลโก้ Valorant หรือ CS2)
  - ไอคอนอุปกรณ์ที่พวกเขาใช้ (เช่น โชว์รูปเมาส์ที่ใช้ให้เห็นทันทีจากหน้าค้นหา)
